# BeatMinesweep

Autoplayer for a canvas Minesweeper variant.

## Images

![Screenshot 1](/AlmostComplete.png)

![Screenshot 2](/Completion.png)

## Use

- Load the folder as an unpacked Chrome extension
- Open the game
- Toggle with `Ctrl+Shift+M`

## Files

- `autoplayer.js` main logic
- `popup.html` UI
- `popup.js` UI logic
- `background.js` service worker
- `manifest.json` extension manifest

## Notes

- Medium mode only (1080x840 canvas)
